
> A Vue.js project by zghong


## feature
1. vue + vue-router + vuex + webpack(vue全家桶)
2. 基于 vue-cli,重新组织代码结构,规范开发流程
3. 集成 mint-ui 基础组件库
4. 集成 lodash ,便于前端处理负责的数据结构
5. 引入 vue-i18n ,增加模板翻译功能
